# Askleion — Branding Pack v1.0

## Concepto
Columna griega estilizada con una forma central abstracta inspirada en Asclepio.
La forma central evita representar una serpiente de manera literal.

## Paleta
- Azul profundo: #0D1B2A
- Crema: #F5F2EB
- Dorado: #CDAA63

## Uso recomendado
- Sidebar: símbolo o logo horizontal.
- Login: logo vertical.
- PWA: iconos cuadrados sin texto.
- Fondo claro: versiones azules.
- Fondo oscuro: versiones crema.
- No deformar, inclinar, recolorear ni agregar sombras.
